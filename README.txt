# MultiDimensionsGame

You will find all the folders and files that make up this video game. This game allows the player to move in 2D and 3D as needed.
was created with Unity in C#.

By downloading the "running game" folder you will simply have to launch the . exe to play.
If you want to change the game from Unity you can download "Unity editoring game".
Hopefully you will have fun in any case on this game.
